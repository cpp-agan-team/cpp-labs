# cpp-labs

> 一组用 C++ 实现的 Linux 编程小组件 —— 把一些底层机制带大家一个个吃透，既能提高自己的底层实力，又能真正用于排障，也能为简历攒下有含金量的技术栈。

## 这个仓库是干什么的

`cpp-labs` 用一系列**可独立完成、可演示、可写进简历**的C++ 小组件，系统性地理解 Linux底层能力。

每个子项目都是**完全独立的 CMake 工程**，在各自目录下 `cmake -B build` 即可单独构建，
互不依赖。
